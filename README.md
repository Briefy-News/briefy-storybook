# briefy storybook

🔗 _**<a href="https://briefy-storybook.vercel.app">storybook 배포 링크</a>**_

## Installation
```bash
npm i briefy-storybook
```

## Usage

```jsx
import { Button } from 'briefy-storybook'

function MainPage() {
  return (
    <Button
      onClick={() => alert('click!')}
      title="name"
      size="medium"
      mode="lineBlue"
    />
  )
}
```
